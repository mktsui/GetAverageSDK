# Get Average

This project is a test to use Kotlin library to find average of entered numbers

## Compatibility
minSDKversion = 9
targetSDKVersion = 28

## Usage

### add numbers
example:
```
getAverage.addMoreNumbers([Integer])
```

### find average
example:
```
getAverage.computeAverageInt()
```
